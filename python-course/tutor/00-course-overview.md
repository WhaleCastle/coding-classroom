# Python Course — Overview (for the tutor)

A complete beginner course for a Year 8 student. No previous coding experience
assumed. Every chapter ends with a small **product** the student builds himself,
so he always finishes with something that works.

## How to use this course

- Each chapter is one file: `01-hello-python.md`, `02-...` and so on.
- Follow the chapter script **in order, one step at a time**. Each step says what
  to explain, what to ask the student to do, and what success looks like.
- Chapters get gradually harder. Never skip the gate conditions at the end of
  each chapter.
- The student saves his work in `../student/chapter-XX/`.

## Difficulty rules (apply across the whole course)

- Chapters 1–4: tiny steps, lots of repetition, instant wins.
- Chapters 5–9: steps get slightly bigger; student should start predicting what
  code will do before running it ("What do you THINK will happen?").
- Chapters 10–14: tutor explains less, asks more; student plans small pieces
  himself. Still: hints, never solutions.
